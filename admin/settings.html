Cara Membuat Bin Baru di JSONBin.io

  1. Kunjungi jsonbin.io dan buat akun (atau login)
  2. Klik tombol "Create New Bin" untuk membuat bin baru
  3. Isi nama bin (opsional) dan klik "Create Bin"
  4. Di halaman bin, klik icon "Secret Key" untuk melihat API Key Anda
  5. Salin BIN_ID dari URL (contoh: jsonbin.io/69b119d4d746013455b0a8bc/...) dan API Key
  6. Isi formulir di atas dengan BIN_ID dan API_KEY tersebut
